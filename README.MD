## run scrapy
```bash
$ scrapy crawl quotes
```

### storing to json
```bash
$ scrapy crawl quotes -o items.json
```

### storing to csv
```bash
$ scrapy crawl quotes -o items.csv
```

### storing to xml
```bash
$ scrapy crawl quotes -o items.xml
```

### create sqlite3 database (database.py on quotetutorial directory)
```bash
python database.py
```
